== Description==

### **Simple & Lightweight WordPress plugin to implement AJAX Search on WordPress Website.**
Use a custom short code to implement AJAX Searching or Instant Live Searching on WordPress website.

Get instant results, the name of the post, the excerpt of that post if applicable. Or the title of any page that matches the search keyword.

### **Where to find the Short Code**
Simply install the plugin & activate it. Then navigate to 'Hound' menu option below posts on the dashboard panel. You will find a short code and a button to copy it.

### **How to customize the search form.**
Just navigate to Hound > Settings, you will find options to edit the search form. The names are quite intuitive, it is very easy to customize.

### **Beautiful Options Panel.**
The options panel is very beautiful and easy to use. You will find all the options in one place. You can customize the search form, search results, and many more.
It is built with [CodeStar Framework](https://github.com/Codestar/codestar-framework), which is a lightweight yet very powerful framework for WordPress plugins.
The plugin is 100% GPL. It has been integrated google fonts via CDN.

== Installation ==

The usual, automatic way;

1. Open WordPress admin dashboard, go to Plugins, click Add New
2. Enter "Hound - AJAX Search Lite" in search and hit Enter
3. Plugin will show up as the first on the list, click "Install Now"
4. Activate & open plugin's settings page located in WordPress admin dashboard, Media

Or if needed, install manually;

1. Download the plugin.
2. Unzip it and upload to _/wp-content/plugins/_
3. Open WordPress admin - Plugins and click "Activate" next to the plugin
4. Activate & open plugin's settings from Media > Increase Maximum Upload Limit
